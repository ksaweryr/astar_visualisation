# A* visualisation
A small demo created for The 5th High School in Krak√≥w open day 2022 using C++17 and SDL2.

## Build prerequisites:
- cmake
- a C++ compiler with C++17 support
- GNU make
- bash
- GNU coreutils
- sed
- xxd
- ImageMagick v6 (or newer)
- SDL2 development package
- SDL2_ttf development package

## Building:
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
